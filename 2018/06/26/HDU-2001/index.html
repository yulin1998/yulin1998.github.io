HDU 2001

发表于 2018-06-26 | 分类于 HDU

Problem Description

输入两点坐标（X1,Y1）,（X2,Y2）,计算并输出两点间的距离。

Input

输入数据有多组，每组占一行，由4个实数组成，分别表示x1,y1,x2,y2,数据之间用空格隔开。

Output

对于每组输入数据，输出一行，结果保留两位小数。

Sample Input

0 0 0 1
0 1 1 0

Sample Output

1.00
1.41
